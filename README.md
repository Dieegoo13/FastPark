<h2 id="sobre">📋 Sobre o Projeto</h2>

O **FastPark** é um sistema criado para simular o funcionamento de um estacionamento real, controlando o fluxo de veículos através da geração de tickets.

O projeto foi desenvolvido com o objetivo de praticar:

- Arquitetura **MVC na prática**
- Organização de rotas personalizadas
- Integração entre **back-end e front-end**
- Manipulação de banco de dados com **MySQL**
- Separação de responsabilidades (Controllers, Models e Views)
- Desenvolvimento de sistemas administrativos reais

O sistema calcula automaticamente o valor do estacionamento com base no tempo de permanência do veículo.

<h2 id="funcionalidades">🚀 Funcionalidades Principais</h2>

- 🎟️ **Geração automática de tickets**
- 🚗 **Registro de entrada de veículos**
- ⏱️ **Controle de tempo de permanência**
- 💰 **Cálculo automático do valor a pagar**
- ✅ **Registro de saída do veículo**
- 📊 **Dashboard com informações do estacionamento**
- 🧱 **Estrutura organizada em MVC**
- ⚡ **Interface simples e funcional**
